# DataPig

## 克隆仓库：
 ```bash
   git clone https://github.com/ziqing888/DataPig.git
   cd DataPig
   ```
## 安装依赖项：
  ```bash
  npm install
  ```
## 在项目根目录中创建名为 private_keys.txt 的文件。REF_CODE 是推荐您账户的代码。文件内容格式如下：
```bash
<PRIVATE_KEY_1>,<REF_CODE_1>
<PRIVATE_KEY_2>,<REF_CODE_2>
```
运行脚本
完成设置后，可以使用以下命令运行机器人：
```bash
node index.js
```
